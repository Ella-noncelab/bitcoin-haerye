# Bitcoin Haerye (비트코인 해례본)

> **해례** 解例 보기를 들어서 풀이함

이 프로젝트는 비트코인 관련 기술 자료의 한글화 프로젝트입니다.

영문 자료 번역 뿐 아니라 기술 이해를 돕기 위한 정보가 더 필요한 부분에 해석을 덧붙이는 것을 목표로 합니다.
